# Natural Language Processing Projects

> Classical NLP, Word Embeddings, Named Entity Recognition, Neural Machine Translation, Attention Mechanisms, and Transformers.

A collection of Natural Language Processing projects completed during the NLP course at Alexandria University.

The projects cover the progression of NLP techniques from traditional statistical methods to modern deep learning architectures.

## Repository Structure

### 1. Language Modeling and Text Classification

Implementation of classical NLP techniques for language modeling and document classification.

**Topics**

* N-Gram Language Models
* Text Generation
* Naive Bayes Classification
* Logistic Regression
* Model Evaluation

**Datasets**

* WikiText-2
* AG News

---

### 2. Word Embeddings and Named Entity Recognition

Learning distributed word representations using Word2Vec and applying them to Named Entity Recognition (NER).

**Topics**

* Word2Vec
* Skip-Gram with Negative Sampling
* Word Analogies
* Feed-Forward Neural Networks
* Hidden Markov Models
* Named Entity Recognition

**Dataset**

* CoNLL-2003

---

### 3. Neural Machine Translation

Implementation of neural sequence-to-sequence models for French-to-English machine translation.

**Topics**

* Transformer Architecture
* Multi-Head Attention
* Seq2Seq Learning
* Bidirectional LSTMs
* Attention Mechanisms
* Beam Search Decoding
* BLEU Evaluation

**Dataset**

* Parallel French-English Translation Corpus

---

The projects are organized to demonstrate the evolution of NLP approaches:

**Statistical NLP → Word Representations → Deep Learning & Transformers**

Each project contains its own README with implementation details, methodology, results, and evaluation.
